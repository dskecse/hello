# Hello

A 'hello world' package for Rust.

## Installation

To depend on a library, add it to your `Cargo.toml`.
Pull in that library using `extern crate` in your `main.rs`.

## Usage

Here's a simple program using `hello`:

```rust
extern crate hello;

fn main() {
    hello();
}
```

You can simply run it with:

    $ cargo run
